# Employee Attrition Analysis & Prediction

## Objective
Predict employee attrition using machine learning.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Scikit-learn

## Models
- Logistic Regression
- Random Forest
- SVM
- XGBoost

## Results
- Achieved ~85% accuracy
- Identified key factors like overtime and job role
